# Support

## Getting Help

Welcome to the `@deessejs/fp` community! Here are some ways to get help:

### Questions & Discussions

- **GitHub Discussions**: For general questions and community discussions
  - URL: https://github.com/nesalia-inc/core/discussions

### Reporting Issues

If you find a bug or have a feature request:

1. Check if the issue already exists
2. Create a new issue with:
   - Clear title describing the problem
   - Detailed reproduction steps
   - Expected vs actual behavior
   - Environment details (Node.js version, OS, etc.)

### Labels to Use

When creating issues, please apply appropriate labels:

- **Type**: `bug`, `feature`, `enhancement`, `docs`
- **Priority**: `priority:critical`, `priority:high`, `priority:medium`, `priority:low`
- **Effort**: `effort:xs`, `effort:s`, `effort:m`, `effort:l`, `effort:xl`

## Security Issues

For security vulnerabilities, please **do not** create a public issue.

Email: **security@nesalia.com**

We will acknowledge your report within 24 hours and provide a timeline for resolution.
