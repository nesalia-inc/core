This is a minimal API reference. For detailed explanations, see the Guides and Concepts sections.

## Creating

| Function | Description |
|----------|-------------|
| `ok(value)` | Create successful Result |
| `err(error)` | Create failed Result |
| `some(value)` | Create present Maybe |
| `none()` | Create absent Maybe |
| `fromNullable(value)` | Convert null/undefined to Maybe |
| `success(value)` | Create successful Outcome |
| `cause(options)` | Create business error Outcome |
| `exception(options)` | Create system error Outcome |
| `okAsync(value)` | Create successful AsyncResult |
| `errAsync(error)` | Create failed AsyncResult |
| `attempt(fn)` | Wrap sync function that might throw |
| `attemptAsync(fn)` | Wrap async function |

## Transforming

| Function | Description |
|----------|-------------|
| `map(result, fn)` | Transform the value |
| `flatMap(result, fn)` | Chain operations that return Result |
| `mapErr(result, fn)` | Transform the error |

## Extracting

| Function | Description |
|----------|-------------|
| `getOrElse(result, default)` | Get value or default |
| `getOrCompute(result, fn)` | Compute default lazily |
| `match(result, ok, err)` | Pattern matching |
| `tap(result, fn)` | Side effects without changing value |
| `tapErr(result, fn)` | Side effects on error |

## Type Guards

| Function | Description |
|----------|-------------|
| `isOk(result)` | Check if Ok |
| `isErr(result)` | Check if Err |
| `isSome(maybe)` | Check if Some |
| `isNone(maybe)` | Check if None |
| `isSuccess(outcome)` | Check if Success |
| `isCause(outcome)` | Check if Cause |
| `isException(outcome)` | Check if Exception |

## Converting

| Function | Description |
|----------|-------------|
| `toResult(maybe, onNone)` | Maybe → Result |
| `toMaybe(result)` | Result → Maybe |
| `toOutcome(result)` | Result → Outcome |

## Async Utilities

| Function | Description |
|----------|-------------|
| `all(...results)` | Run multiple AsyncResults in parallel |
| `race(...results)` | First AsyncResult to resolve |
| `traverse(items, fn)` | Map array to AsyncResults |
| `retryAsync(fn, options)` | Retry failed async operations |
| `sleep(ms)` | Async delay |

## Options & Config

### RetryOptions

```typescript
interface RetryOptions {
  attempts?: number;      // Default: 3
  delay?: number;         // Initial delay in ms, default: 1000
  backoff?: "exponential" | "linear" | "constant";
  predicate?: (error: Error) => boolean;
  onRetry?: (error: Error, attempt: number) => void;
  jitter?: boolean;
}
```

## Quick Lookup

Looking for something specific?

- **Validation** → Use `ok()` + `err()` + `Result`
- **Optional values** → Use `some()` + `none()` + `Maybe`
- **Parsing JSON** → Use `attempt()` + `Try`
- **Network calls** → Use `AsyncResult`
- **Rich errors** → Use `Outcome`

See [Choosing a Type](/docs/concepts/choosing-type) for detailed guidance.
