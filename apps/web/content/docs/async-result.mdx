# AsyncResult

The `AsyncResult` type handles asynchronous operations with proper error typing. It wraps promises and provides the same functional patterns as `Result`.

## Overview

```typescript
type AsyncResult<T, E = Error> = Promise<AsyncResultInner<T, E>>;

type AsyncResultInner<T, E> = AsyncOk<T> | AsyncErr<E>;
```

- **AsyncOk<T>** - Successful async result with value `T`
- **AsyncErr<E>** - Failed async result with error `E`

## Creating an AsyncResult

### okAsync - Create Async Success

```typescript
import { okAsync } from "@deessejs/core";

const result = await okAsync(42);
// AsyncOk { ok: true, value: 42 }
```

### errAsync - Create Async Failure

```typescript
import { errAsync } from "@deessejs/core";

const result = await errAsync("Something went wrong");
// AsyncErr { ok: false, error: "Something went wrong" }
```

### fromPromise - Wrap a Promise

```typescript
import { fromPromise } from "@deessejs/core";

const result = await fromPromise(
  fetch("/api/data").then(r => r.json())
);
```

### fromPromise - Handle Rejections

```typescript
import { fromPromise, isOk } from "@deessejs/core";

const result = await fromPromise(Promise.reject(new Error("fail")));

console.log(isOk(result)); // false
console.log(result.error.message); // "fail"
```

## Transforming AsyncResults

### map - Transform the Value (Sync)

```typescript
import { okAsync, map } from "@deessejs/core";

const result = await okAsync(10)
  .then(map(x => x * 2));

console.log(result.value); // 20
```

### mapAsync - Transform with Async Function

```typescript
import { okAsync, mapAsync } from "@deessejs/core";

const result = await mapAsync(
  okAsync(10),
  async (x) => x * 2
);

console.log(result.value); // 20
```

### flatMap - Chain AsyncResults

```typescript
import { okAsync, flatMap } from "@deessejs/core";

const result = await okAsync(10)
  .then(flatMap(x => okAsync(x * 2)));
```

### flatMapAsync - Chain with Async Function

```typescript
import { okAsync, flatMapAsync } from "@deessejs/core";

const result = await flatMapAsync(
  okAsync(10),
  async (x) => okAsync(x * 2)
);
```

## Handling AsyncResults

### getOrElse - Get Value or Default

```typescript
import { okAsync, errAsync, getOrElse } from "@deessejs/core";

await okAsync(10).then(r => getOrElse(r, 0)); // 10
await errAsync("e").then(r => getOrElse(r, 0)); // 0
```

### getOrCompute - Compute Default Lazily

```typescript
import { errAsync, getOrCompute } from "@deessejs/core";

const result = await errAsync("e")
  .then(r => getOrCompute(r, async () => 42));
```

### match - Pattern Matching

```typescript
import { okAsync, match } from "@deessejs/core";

const result = await match(
  okAsync(42),
  (value) => `Success: ${value}`,
  (error) => `Error: ${error}`
);
```

### tap - Side Effects

```typescript
import { okAsync, tap } from "@deessejs/core";

await okAsync(10).then(tap(value => console.log(value)));
```

## Parallel Operations

### all - Run Multiple AsyncResults in Parallel

```typescript
import { okAsync, all } from "@deessejs/core";

const results = await all(
  okAsync(1),
  okAsync(2),
  okAsync(3)
);

console.log(results); // [1, 2, 3]
```

### race - First Result to Complete

```typescript
import { okAsync, race } from "@deessejs/core";

const fast = okAsync(1);
const slow = new Promise(r => setTimeout(() => r({ ok: true, value: 2 }), 100));

const result = await race(fast, slow);
// Returns the first to resolve
```

### traverse - Map Over Array

```typescript
import { okAsync, traverse } from "@deessejs/core";

const items = [1, 2, 3];
const result = await traverse(items, async (x) => okAsync(x * 2));

console.log(result); // [2, 4, 6]
```

## Converting

### toNullable - Convert to Nullable

```typescript
import { okAsync, toNullable } from "@deessejs/core";

await toNullable(okAsync(42)); // 42
await toNullable(errAsync("e")); // null
```

### toUndefined - Convert to Undefined-able

```typescript
import { okAsync, toUndefined } from "@deessejs/core";

await toUndefined(okAsync(42)); // 42
await toUndefined(errAsync("e")); // undefined
```

## API Reference

| Function | Description |
|----------|-------------|
| `okAsync<T>(value: T)` | Create async success |
| `errAsync<E>(error: E)` | Create async failure |
| `fromPromise(promise)` | Wrap a promise |
| `map(result, fn)` | Transform value (sync) |
| `mapAsync(result, fn)` | Transform value (async) |
| `flatMap(result, fn)` | Chain (sync) |
| `flatMapAsync(result, fn)` | Chain (async) |
| `getOrElse(result, default)` | Get value or default |
| `getOrCompute(result, fn)` | Compute default lazily |
| `match(result, ok, err)` | Pattern matching |
| `tap(result, fn)` | Side effect |
| `all(...results)` | Run in parallel |
| `race(...results)` | First to resolve |
| `traverse(items, fn)` | Map over array |
| `toNullable(result)` | Convert to null |
| `toUndefined(result)` | Convert to undefined |
