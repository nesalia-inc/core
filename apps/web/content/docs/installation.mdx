Get up and running with @deessejs/core in just a few minutes.

## Install the Package

Install `@deessejs/core` using your preferred package manager:

```bash
# pnpm (recommended)
pnpm add @deessejs/core

# npm
npm install @deessejs/core

# yarn
yarn add @deessejs/core
```

## Requirements

- TypeScript 5.0 or higher
- Node.js 18 or higher

## Quick Start

After installation, import the functions you need:

```typescript
import { ok, err, map, Result } from "@deessejs/core";

// Create a Result - explicit about potential failure
const divide = (a: number, b: number): Result<number, string> => {
  if (b === 0) return err("Cannot divide by zero");
  return ok(a / b);
};

// Use it - TypeScript forces you to handle errors
const result = divide(10, 2)
  .map(x => x * 2)
  .getOrElse(0);

console.log(result); // 5
```

## TypeScript Configuration

No special TypeScript configuration is required. The library exports full type definitions out of the box.

For the best experience, ensure your `tsconfig.json` has strict mode enabled:

```json
{
  "compilerOptions": {
    "strict": true
  }
}
```

Strict mode enables:
- Strict null checks
- No implicit any
- Strict property initialization

## ESM Support

The package is published as an ES module. Ensure your `package.json` has:

```json
{
  "type": "module"
}
```

If you're using CommonJS, import like this:

```typescript
import { ok } from "@deessejs/core/index.js";
```

## Importing from @deessejs/core

### Individual Imports (Recommended)

Import only what you need for smaller bundles:

```typescript
import { ok, err, map, getOrElse } from "@deessejs/core";
```

### All Exports

Import everything if needed:

```typescript
import * as Deesse from "@deessejs/core";

Deesse.ok(42);
Deesse.err("error");
```
