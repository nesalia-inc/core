---
title: Retry
description: Retry utilities for resilient operations
---

The `retry` utilities provide resilient patterns for handling transient failures. They automatically retry failed operations with configurable backoff strategies.

## retry - Synchronous Retry

```typescript
import { retry } from "@deessejs/core";

const result = retry(() => {
  // This might fail
  return fetchData();
}, { attempts: 3 });
```

### Basic Options

```typescript
const result = retry(
  () => mightFail(),
  {
    attempts: 3,    // Number of attempts (default: 3)
    delay: 1000,    // Initial delay in ms (default: 1000)
  }
);
```

## retryAsync - Asynchronous Retry

```typescript
import { retryAsync } from "@deessejs/core";

const result = await retryAsync(
  async () => {
    const response = await fetch("/api");
    return response.json();
  },
  { attempts: 5 }
);
```

## Backoff Strategies

### Exponential Backoff (Default)

```typescript
import { retryAsync } from "@deessejs/core";

// Delays: 1000, 2000, 4000...
await retryAsync(fn, { delay: 1000, backoff: "exponential" });
```

### Linear Backoff

```typescript
// Delays: 1000, 2000, 3000...
await retryAsync(fn, { delay: 1000, backoff: "linear" });
```

### Constant Backoff

```typescript
// Delays: 1000, 1000, 1000...
await retryAsync(fn, { delay: 1000, backoff: "constant" });
```

### Custom Backoff Function

```typescript
await retryAsync(fn, {
  backoff: (attempt, delay) => {
    // Custom logic
    return delay * Math.pow(2, attempt) + Math.random() * 100;
  }
});
```

## Jitter - Prevent Thundering Herd

Add random jitter to prevent all retries from happening at once:

```typescript
await retryAsync(fn, { jitter: true });
// Delay becomes: delay * (0.5 + random())
```

## Predicate - Filter Retryable Errors

```typescript
import { retryAsync } from "@deessejs/core";

// Only retry on network errors
await retryAsync(
  async () => mightFail(),
  {
    predicate: (error) => {
      return error.code === "NETWORK_ERROR";
    }
  }
);
```

## onRetry Callback

Execute code before each retry:

```typescript
import { retryAsync } from "@deessejs/core";

await retryAsync(
  async () => mightFail(),
  {
    attempts: 3,
    onRetry: (error, attempt) => {
      console.log(`Retry ${attempt} after error: ${error.message}`);
    }
  }
);
```

## RetryOptions

```typescript
interface RetryOptions {
  /** Number of attempts (default: 3) */
  attempts?: number;

  /** Initial delay in ms (default: 1000) */
  delay?: number;

  /** Backoff strategy */
  backoff?: "exponential" | "linear" | "constant" | ((attempt: number, delay: number) => number);

  /** Predicate to determine if error is retryable */
  predicate?: (error: Error) => boolean;

  /** Callback on each retry */
  onRetry?: (error: Error, attempt: number) => void;

  /** Add jitter (default: false) */
  jitter?: boolean;
}
```

## Backoff Strategies - Export

```typescript
import {
  exponentialBackoff,
  linearBackoff,
  constantBackoff
} from "@deessejs/core";

const delay = exponentialBackoff(attempt: number, delay: number): number;
const delay = linearBackoff(attempt: number, delay: number): number;
const delay = constantBackoff(attempt: number, delay: number): number;
```

## Examples

### Web Request with Retry

```typescript
import { retryAsync } from "@deessejs/core";

const fetchWithRetry = async (url: string) => {
  return retryAsync(
    async () => {
      const response = await fetch(url);
      if (!response.ok) {
        throw new Error(`HTTP ${response.status}`);
      }
      return response.json();
    },
    {
      attempts: 3,
      delay: 1000,
      backoff: "exponential",
      predicate: (error) => {
        // Only retry on network errors
        return error.message.includes("fetch");
      }
    }
  );
};
```

### Database Connection

```typescript
import { retry } from "@deessejs/core";

const connect = retry(
  () => database.connect(),
  {
    attempts: 5,
    delay: 2000,
    backoff: "linear",
    onRetry: (error, attempt) => {
      console.log(`Connection attempt ${attempt} failed`);
    }
  }
);
```

## API Reference

| Function | Description |
|----------|-------------|
| `retry(fn, options)` | Retry sync function |
| `retryAsync(fn, options)` | Retry async function |

| `linearBackoff(attempt, delay)` | Linear delay calculator |
| `constantBackoff(attempt, delay)` | Constant delay calculator |
